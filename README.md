# Tesla-Light-Show
Collection of my custom light shows. 

# Instructions:
1. Download the "LightShow" directory.
2. Copy it in your USB drive (exFat or MSDOS format).
3. Plug it in your tesla. 
4. Toybox > Light Show > (Wait for the custom light show to load) > 
   Get Out and Enjoy!
